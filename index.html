    • Проект от Т-Банка "Языковая модель для генерации подсказок" на Большой математической мастерской в Омске (https://github.com/ruzanovad/bmm-tbank)
      Classic ML
      LLM
      Команде был предоставлен размеченный датасет, который содержал промпт (код на одном из 5 языков программирования), дополнение этого кода некой языковой моделью, каноническое решение (для справки) и целевая переменная, обозначающая, верно ли модель сгенерировала подсказку или нет. В качестве целевой метрики была использована F_1.25 мера.
      Цель проекта — создать модель, способную оценивать качество автоматически сгенерированных подсказок для кода на разных языках программирования (решающую задачу бинарной классификации).
      Что было сделано:
      1. Проведен разведочный анализ данных (EDA), выявлены закономерности и особенности датасета.
      2. Разработаны признаки (feature engineering) для улучшения качества классификации.
      3. Протестированы различные алгоритмы бинарной классификации, включая традиционные методы и нейросетевые архитектуры.
      4. Лучший результат показала fine-tuned модель на основе BERT (с HuggingFace), которая достигла наибольшего значения F_1.25-меры.
      5. Обучение проводилось на Kaggle с использованием Weights & Biases для мониторинга экспериментов.
